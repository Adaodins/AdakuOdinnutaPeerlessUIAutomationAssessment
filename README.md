\# SauceDemo Cypress Automation



This repository contains Cypress automation tests for \[SauceDemo](https://www.saucedemo.com/).



\## Test Coverage

\- Login tests (valid, invalid, locked-out)

\- Add to cart and remove from cart

\- Checkout process (positive and negative scenarios)



\## Setup

1\. Install dependencies: `npm install`

2\. Open Cypress test runner: `npx cypress open`

3\. Run tests in Chrome or your preferred browser



\## Notes

\- `node\_modules` is ignored
